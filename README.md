# p2p_book_sharing

내 책장의 책, 여러분 책장의 책을 나누어봐용.

## 개선해야 할 것.

-콜렉션 모델과 북 모델 연결을 위한 외래키를 book_id가 아닌 isbn13으로.
:북 디비의 내용을 비우고 다시 크롤링 하더라도 관계가 유지 될 수 있도록.
-우리집 책 등록시 책 검색 강화
:느리다. 책 이름 입력시 띄어쓰기 틀리면 검색 안된다.
